# 前衛技術同盟綱領

(a) Anti-Copyrights 2025, 前衛技術同盟. All resources shared

## 序文

前衛芸術家ならぬ我々「前衛技術者」は、技術の持つ政治性や権力性とエンジニア労働の体制再生産を自覚して反体制側に付いた技術者である。価値や意味を扱う芸術よりも、技術の方がよほど社会的に重要なインフラであるのに、何故これまで技術それ自体を政治闘争の主軸に据えなかったのだろうか？芸術は表象である故に政治性と直結しやすいのに対し、技術を道具的理性と純粋な自然科学的中立性を称し、技術の政治性を隠蔽する近代の構造が存在するからだ。我々はこの”技術の脱政治化”を拒否し、技術の権力性を肯定し、積極的なアナキスト社会革命の闘争として前衛技術運動を展開する。

## 第一章　前衛技術論

### 1. 技術の中立性は存在しない  
技術は常に特定の社会的支配関係を物質化する。マルクス主義的に言えば技術とは、下部構造の生産関係そのものではないか。クラウドや検索アルゴリズム、企業の労務管理ソフトや人事評価システム、GIGAスクール構想などデジタル教育、青少年保護を謳ったフィルタリングや検閲は、国家と資本主義と家父長制の支配構造そのものを体現する機構である。我々は「技術は中立な道具」というリベラルな言説の虚構を暴き、技術が我々の存在を強く規定する権力性を持つことを明らかにする。ハイデガーは『技術への問い』において、この近代文明の技術の本質をゲシュテル（総駆り立て体制）と存在論的に批判した。また、メディアやSNS広告は、シチュアシオニストの理論を用いて、スペクタクル（見せかけの表象）と捉えて分析することができる。

### 2. デジタル・レーニン主義の脅威
中共型統治手法、ビックデータ独裁とも目されるデジタル・レーニン主義など技術管理社会の脅威は、西側の資本主義社会にも静かに迫っている。セバスチャン・ハイルマンが提唱した概念である「デジタル・レーニン主義」とは、情報を一元的に当局に収集し中央集権化し、諸個人の反逆をデータ分析に基づいて事前に抑制するゲシュテル機構である。

中共の社会信用システム、GAFAら大手テック企業の監視資本主義、AWSやAzureのクラウド荘園などテクノ封建制、日本のマイナ保険証、イスラエル帝国の軍事用ＡＩ兵器——これらはすべて同一の構造を持ち、ゲシュテルとして統一的かつ根元的な批判を展開できる。資本主義スペクタクル商品社会における**人間が作ったはずの技術に人間が支配される**という、フォイエルバッハのキリスト教批判に酷似した倒錯した疎外形態が、デジタル・レーニン主義の本質である。そう、技術は神となったのだ。

### 3. 既存技術運動への批判と部分的評価
既存の技術運動は、どれも技術を道具的理性としか見ておらず、技術の権力性に無自覚である。自由ソフトウェア運動やオープンソースは、自由な技術を掲げつつも、結局はスペクタクルに回収されている。ただ、自由と協力を重んじるハッカー文化は尊敬する。暗号アナキズムは市場原理主義的で右派的だとして批判に値するが、匿名性を近代的主体の脱構築として転用する。左派加速主義はカウツキーの資本主義自壊論の出来の悪い後継に過ぎず、右派加速主義は単なる能天気な技術楽観主義だ。これらの潮流では資本主義スペクタクルへの根源的批判が出来ず、ゲシュテルの北極＝デジタル・レーニン主義を前に敗れ去っていくだろう。我々はどれにも決して迎合しない、独自の技術論である。

### 4. ハイデガーの左捻り  
ハイデガーが論じたゲシュテル、すなわち人間をも資源として配置する資本主義的技術の本質を、ハイデガーが「存在の思惟」と言ったように受動的に考察する深淵な態度を取るのではなく、我々はハイデガー左派として、総力を上げて積極的かつ徹底的に粉砕する。そして、技術の存在論的可能性、存在の開示としての本来のテクネーとしての技術の本質＝ポイエーシス（詩的創造）に技術を転倒させ、技術が専門職から解放され我々の自由な生と一体化する未来を構想する。技術を解放するには、ピュシス（存在の真理）の開けを見るには、ポイエーシスに回帰するには、ゲシュテルの打倒＝社会革命は不可避である。これがハイデガー左派の解釈である。

我々のテーゼはこうだ。
> エンジニアの廃止こそは技術の実現である

これはギー・ドゥボールの言葉「アーティストの廃止こそ芸術の実現である」の転用である。即ち、エンジニアという資源として挑発されゲシュテルに配置されて、体制再生産に加担させられるという自己疎外から脱却し、技術者が本来性を回復し技術をポイエーシスとして開示させるという意味である。我々は、技術を革命の手段にするのではなく、技術それ自体を革命する、政治的実践による存在論的革命を指向する。 

## 第二章　組織論

### 1. 前衛技術者のエリート主義  
同盟は技術における前衛党であり、前衛技術論の技術哲学への深い理解及び批判的発展能力と、何らかの技術的技能によるラディカルな実践の両面で徹底的に鍛え上げられた少数の**前衛技術者**のみによって構成される。大衆的な「誰でも参加できる」運動は、趣味者やミーハーの流入を許容し、理論の劣化やスペクタクルへの回収を招くだけあり、前衛技術論を堅持することすら困難になる。もちろん、ゲシュテルに疎外された大多数の民衆こそが、真に前衛技術によって解放されるべきであり、彼らに対しては非権威的・水平的・対話的な関係において我々の指導力を発揮し、世人の自己疎外から抜け出し本来性と自由を回復し、自己解放できるように手助けせねばならない。

### 2. 純度の為には除名を厭わない
ハイデガーが「存在の思惟」を必要としたのは、世人の意識に強く内面化されたゲシュテルを根源的に批判するためであった。よって、我々が革命を成すには、高度に抽象化・体系化された理論を要する。我々は技術前衛党として、 高度な前衛技術論を一貫し、自己進化し続けるゲシュテルの形態に応じて理論に批判的発展を加えていくことは、ゲシュテルへの批判を継続的に展開する上の当然である。理論を歪曲した者、ゲシュテルへ妥協した者、敵に寝返った者は即座に除名される。これはシチュアシオニスト・インターナショナル伝統の厳格な組織原理の継承である。

### 3. 左右混交のラディカリズム
我々は体制に安住して偽善の技術倫理を説くリベラル派を心底軽蔑する。アナキストやマルクス主義者、サンディカリスト、ファシストや国民社会主義者など、極左や極右に分類されるラディカル派、過激な反体制派をオルグする。我々が為すべきは存在論的革命であり、政治的ドクトリンの違いで分裂するような世俗性は排除する。存在論的な視点から技術管理社会を分析し超克する必要があるため、リベラルな技術倫理に囚われないラディカル派こそが「ゲシュテル粉砕」を掛け声に結束せねばならない。ゲシュテルが技術文明において普遍的に散見される疎外形態であるからして、我々の革命も普遍的な対抗策を打ちださなければならないのだ。なお、組織拡大は、我々の側から新規構成員をオルグするのであって、たとえ誰かが参加希望を出しても加入は認めない。

## 第三章 革命実践

### 1. 転用によるスペクタクルの撹乱
我々の力だけでは全てを為し得ない。既存の技術を積極的に転用(Détournement)し、資本主義スペクタクルを撹乱する事こそ、陣地戦＝ヘゲモニー闘争であり、一定のシンパ層の強い支持を獲得するための、我々の最初の革命任務である。公式サイトやこの綱領文書などというゲシュテルの前提にとっての”異物”を無許可でチャット——もちろん大真面目なゲシュテルである会社や学校のSlackのスレッドなど——に投下するなど、レトリスト／シチュアシオニストの戦術を踏襲する。

### 2. ゲシュテルへの反逆と破壊
ゲシュテルの管理に公然と反逆すること、場合によっては物理的に破壊することによって、初めてゲシュテルは文字通り粉砕される。これは、弾圧リスクが極めて高いので、組織だった行動と救援体制が求められる。故に、ヘゲモニー闘争によってシンパ層を組織内において二割程度か獲得できた上で実行される方が無難だ。労務管理ソフトの破壊的クラックや、資本主義的アプリの本番環境のデータベースの全削除、貸与されたGIGAスクール構想のパソコンを全部上書きしてLinuxをぶち込むなど、これ以上は綱領に書けないような手段を使ってでも、この根深いゲシュテルを総破壊する。

### 3. ポイエーシスとしての技術の構築
破壊への情熱は創造への情熱でもある。こうバクーニンが述べたように、ゲシュテルを打倒した上に普遍化されるポイエーシスとしての技術を構築せねばならない。それは技術が疎外されず、日常生活の中に溶け込んだ様式であり、エンジニア労働は廃止され、誰であれ技術を完全に自らで思いのままの支配し、自らの自由と利益のために技術を奉仕させることができる「状況の構築」こそ技術の実現である。

アーキテクチャの単純化が技術の民主化を可能にするからして、子供騙しのビジュアルプログラミング言語の玩具ではなく、ゲシュテルが持つ複雑性やブラックボックス性を排し、容易でありながらポイエーシスを保持した技術形態として、既存のソフトウェアを転用し改良するか、必要に応じて再発明する。この技術の再発明は、社会の再発明、社会革命を意味する。

これらの実践により、我が同盟の前衛技術論は単なる技術評論ではなく社会的インパクトを持つものになるだろう。
